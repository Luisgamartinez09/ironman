## Run Heart Rate Zones (Fix This Now)
Your long runs are at 167–172 bpm — that's Zone 5 (race effort). Long runs must be Zone 2.

| Zone | Name | HR Range | Use For |
|---|---|---|---|
| Z1 | Recovery | <130 bpm | Warm up, cool down |
| Z2 | Aerobic | 130–145 bpm | Long runs, easy swims |
| Z3 | Tempo | 145–158 bpm | Tempo runs |
| Z4 | Threshold | 158–170 bpm | Intervals |
| Z5 | Max | 170+ bpm | Short sprints only |

**Long run rule: if HR goes above 155, walk until it drops back down.**

## Bike Plan (Once You Have a Bike)
- Weeks 1–8: Gym stationary, build aerobic base, 1–1.5h sessions
- Weeks 9–16: Outdoor rides START. Build from 1.5h → 3h. Focus on smooth pedaling.
- Weeks 17–22: Long rides 4–5h. Practice nutrition on bike.
- Week 23–24: One 5h ride as final long effort.
- Race: Target 20mph avg, RPE 6/10 the whole way. Save legs for the run.

## Nutrition Notes
- Start practicing race nutrition on all rides/long runs NOW
- Target: 60–90g carbs/hour on bike, 45–60g/hour on run
- Train your gut — use gels, bars, sports drink in training
